Introduction:

In one of my courses, we had a lecture on zombies where the professor showcased a SIR model for zombie-ism. It was an interesting model, though I think it has some flaws. Considering I know how mathematical modeling works in ecology and have some experience working with these models (both on paper and using code), I thought it would be a fun project to work on.

First, let me introduce you to the paper. In this paper, they discussed 4 models with different assumptions (their base model, a latent model, a model with quarantine, and a model with treatment). From this, they found that humanity will eventually all become zombies in every model, and the only way to prevent a zombie outbreak is by killing them all by “strategically destroying them”.

For this blog, I will discuss and modify 3 of their models (the base model, the latent infection model, and the model with treatment). I will then run simple simulations each of my models to see how long Toronto will last and discuss potential limitations of these simple models.

The base model:

The paper’s base model

Like in the typical SIR model, the authors divide the population into 3, susceptible (the living), the removed class (the dead), and zombies (the living dead). Susceptibles can only become a zombie when they come into contact with a zombie, and the dead are sometimes resurrected as zombies. They assumed that the birth rate is constant, and that zombies move to the removed class once they are defeated by a susceptible (such as through decapitation). This is their first proposed model, as drawn in the diagram below.

# Here, I will re-create their 1st model (which they did not plot in the paper)

zombie_equations <- function(time, variables, parameters) {
  with(as.list(c(variables, parameters)), {
    dS <- pi - beta * S * Z - delta * S
    dZ <- beta * S * Z + zeta * R - alpha * S * Z
    dR <- delta * S + alpha * S * Z - zeta * R
    
    return(list(c(dS, dZ, dR)))
  }) }

parameters <- c(
  pi = 0,
  alpha = 0.005, 
  beta = 0.0095, 
  zeta = 0.0001, 
  delta = 0.0001 )

initial_values <- c(
  S = 500, 
  Z =   0, 
  R =   0  )

time_values <- seq(0, 15)

zombie_value_1 <- ode(
  y = initial_values,
  time = time_values,
  func = zombie_equations,
  parms = parameters )

zombie_1 <- as.data.frame(zombie_value_1) %>%  
  pivot_longer(-time, names_to = "compartment", values_to = "count")
zombie_1 %>% ggplot(aes(time, count, colour = compartment)) + geom_line() + 
  labs(title = "The outcome of the 1st model in the paper", x = "Time (days)", y = "Count")

My Modified Base Model

However, I think the model has 3 flaws.

For one, as one of my classmates pointed out during lecture, the zombies that had their heads chopped off can still be resurrected from the removed class, which shouldn’t be the case.

Additionally, the model assumes constant births, which I don’t think is realistic. After all, new humans don’t spontaneously pop into existence, they have to be birthed by another human, so the number of births should be a function of the number of susceptible humans. (In the paper, they assumed \(\pi = 0\), which is why I don’t think their base model was super insightful. After all, if there are no additional living people added to to system, then naturally everyone will become a zombie eventually.)

Lastly, in some pop culture media, zombies are subject to natural decay. Plus, they aren’t the brightest bunch, leading them to accidentally offing themselves (such as by drowning or walking off a cliff). So they should be subject to a low background rate of removal that is unrelated the number of susceptibles.

To rectify this, I have made 3 modifications to their model. I assume that - the group of culled zombies cannot be revived - births is a linear function of the susceptible population - there is some low rate of natural zombie removal.

Leading to the equations

\[S' = \pi S - \beta SZ - \delta S \\ Z' = \beta SZ + \zeta R - \alpha SZ - \eta Z \\ R' = \delta S - \zeta R \]

\[S' = \pi S - \beta SZ - \delta S \] \[ Z' = \beta SZ + \zeta R - \alpha SZ - \eta Z \] \[ R' = \delta S - \zeta R \] Looking at the equations, it can be immediately seen that \(S = 0\) is an equilibrium, where \(R = 0\) and \(Z = 0\). Intutitively, this should make sense. If everyone is a zombie, there will be no more children being born and no more people to infect. Since the recovered can only become zombies and zombies cannot go into the recovered class, eventually all recovered will become zombies. Since zombies perish at some natural low rate \(\eta\), all zombies will die off eventually.

Please note: in the paper, they used the lower case \(\xi\). Here, it has been changed to its upper case \(\Xi\) due to its resemblance to \(\epsilon\)

Here, in addition to the same 3 changes made to the previous models, I will use \(\Xi ZS\) instead of \(\Xi Z\). This is to model the fact that susceptibles are needed in the discovery, manufacturing, and delivery of the treatment.
